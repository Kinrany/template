### Stack

[![Greenkeeper badge](https://badges.greenkeeper.io/Kinrany/template.svg)](https://greenkeeper.io/)

 * Typescript

### Commands

```bash
# start a dev server
npm run dev

# run tests
npm run test

# format all code
npm run lint-fix

# build for production
npm run build
```
